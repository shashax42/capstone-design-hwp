# 얼굴 인식 기반 캐리커처 이모티콘 생성 플랫폼 

## 📌 프로젝트 소개
얼굴 인식을 통해 캐리커처 스타일 이모티콘을 자동 생성하고, 나만의 아이덴티티를 표현할 수 있는 맞춤형 플랫폼입니다. 이 레포지토리는 프로젝트의 핵심 코드와 주요 자료를 포함하고 있으며, 전체 코드는 용량 제한으로 인해 업로드되지 않았습니다.
